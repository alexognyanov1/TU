HTML Basics

HTML (HyperText Markup Language) is used to structure web pages. Common elements are headings, paragraphs, links, lists, images, tables, and forms.

  • <h1>...</h1> for titles and headings.
  • <p>...</p> for paragraphs of text.
  • <a href="..."> for navigation links.
  • <img src="..." alt="..."> for images with descriptions.
  • <ul> / <ol> for unordered and ordered lists.
  • <table> and <form> for data and user input.
